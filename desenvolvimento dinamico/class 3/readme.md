#Tipos de operadores
1) Binários e Unários
    0 e 1
    i++ ou i--

2) Aritiméticos
    Adição
    Subtração
    Multiplicação
    Divisão
    Resto ou Módulo
    Exponêncial

3) Incremento e Decremento
    x++
    x--

    var i = 1, j = ++i; //j = 2 e i = 2

    var i = 1, j = i++; //i = 2 e j = 1

4) Comparação
    ==  (igual)
    !=  (diferente)
    === (totalmente igual)
    !== (totalmente diferente)
    >   (maior que)
    >=  (maior ou igual)
    <   (menor que)
    <=  (menor ou igual)

5) Lógico
    AND (&&)
    OR  (||)
    NOT (!)

6) Ternário
    // condicao ? valor1 : valor2
    
    let status = (age >= 18) ? "Adulto" : "Menor de idade";
